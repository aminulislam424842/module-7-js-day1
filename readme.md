# 🚀 JavaScript Fundamentals Practice

A collection of beginner-friendly JavaScript practice programs created while learning the fundamentals of JavaScript through the **Programming Hero AI Driven Full Stack Web Engineering** course.

This repository covers JavaScript basics including variables, data types, arithmetic operations, data conversion, shorthand operators, and simple mini projects.
